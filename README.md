# 🧩 Services Section باستخدام CSS

مشروع Front-End بسيط لإنشاء قسم خدمات (Services Section) كما يوجد في أغلب مواقع الشركات.

الهدف من المشروع هو التدريب على تنظيم الكروت باستخدام Flexbox وإضافة تأثيرات Hover لجعل التصميم أكثر تفاعلية.

---

## 🚀 فكرة المشروع

يتكون المشروع من:

- عنوان للسكشن
- 3 كروت للخدمات
- أيقونة لكل خدمة
- عنوان ووصف
- تأثير Hover يرفع الكارت للأعلى

هذا النوع من السكاشن يستخدم كثيرًا في صفحات الهبوط (Landing Pages).

---

## 🛠️ التقنيات المستخدمة

- HTML5
- CSS3
- Flexbox
- Hover Effects

## 🧠 ماذا تعلمت من المشروع؟

- تنظيم الكروت باستخدام Flexbox
- إنشاء سكشن خدمات احترافي
- استخدام خاصية gap لتنظيم المسافات
- إضافة تأثير Hover على الكروت
- استخدام transform لعمل حركة للكارت
- جعل الحركة ناعمة باستخدام transition
